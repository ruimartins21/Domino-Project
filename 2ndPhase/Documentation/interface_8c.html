char *  pattern,
IDS *  sequenceIds,
unsigned long *  numberOfMatches 
)

This function will search a given pattern among all the sequences generated At the same time, each sequence that contains the pattern, its id will be stored in a structure of ids to have access of the matched sequences only at the other functions The search functionality used here is based on the KMP substring search algorithm

Parameters
allSequencesis the structure that stores all the sequences generated
patternis the pattern chosen to be searched
sequenceIdsis the structure where it will be stored the ids of the sequences matching the patten
numberOfMatcheswill tell the user how many sequences matched with the pattern
